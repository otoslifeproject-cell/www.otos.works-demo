## How to use

1. Open each `.html` in a browser (they reference `otos-continuity-shared.css` in the same folder).
2. For **slides**: use the browser’s print dialog → “Save as PDF” or “Print to PDF”, or take a screenshot of the diagram area.
3. Styling is shared so fonts, colours, and box styles match across all three.
