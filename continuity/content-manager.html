Who sees what you build?

Save & apply only stores hero settings in this browser (localStorage). Visitors to otos.works do not use that. They get whatever is in the HTML file on the server — the FS_HOME_DEFAULTS block in full-scroll2.html.

To make the world match your preview: click Copy for public site, then paste into the repo, commit the image file if new, bump the page rev, push, and deploy.
