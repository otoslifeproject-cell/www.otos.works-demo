# OTOS Hero Image Recreation Briefs

**Purpose:** Complete recreation briefs for every OTOS hero image. Regenerate clean — no typos, no wrong logo. Each image uses fresh language not repeated elsewhere. Each tells the problem and solution from a different angle.

**Hand to:** Midjourney / DALL·E 3 / Adobe Firefly for photos.  
**Glassmorphism overlays:** Built separately in Cursor/HTML (see `hero-overlays/`).

---

## Global Style Rules

| Attribute | Spec |
|-----------|------|
| Photo style | Cinematic photorealism. Shallow depth of field. Real UK faces — lived-in, not stock. Warm but not saccharine. |
| Overlay logo | **OTOS orange brain logo** (amber-to-orange gradient, anatomical but minimal) + **OTOS** wordmark in bold white. Centred top. NOT the drop logo. |
| Headline font | Inter Bold or similar. White. Large. Left-aligned on landscape, centred on portrait. |
| Sub-copy font | Inter Regular. White at 80% opacity. Smaller. Same alignment as headline. |
| CTA button | Pill shape. Dark teal `#0D3D3D` fill. White text. No harsh borders. |
| Stats overlay | Glassmorphism card — see `hero-overlays/hero-overlays.css`. Pinned right on landscape, bottom on portrait. |
